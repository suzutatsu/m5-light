# M5StickC Plus2 PIR & RGB LED Controller (RTC連動・省電力版)

M5StickC Plus2 に接続した PIR（人感）センサーが動きを検知すると、RGB LED が「炎（キャンドル）」のようなエフェクトで点灯するプロジェクトです。

モバイルバッテリーや外部電源による長期間の自動運用を想定し、**指定した時間帯以外は即座に超低消費電力スリープ（Light Sleep）に戻る画期的な省電力化**を行っています。

---

## 主な機能

- **夜間限定点灯 (JST 21:00 〜 翌朝 5:00)**: 
  - 内蔵のリアルタイムクロック（RTC）を利用し、指定した時間帯のみLEDが点灯します。
  - **日中の省電力対策**: 昼間（5:00 〜 21:00）に人感センサーが動きを検知しても、LEDを1ミリ秒も点灯させずに**即座にスリープ状態へ復帰**します。これにより無駄な電力消費を完全にカットし、外部バッテリーの持ちを飛躍的に延ばします。
- **全自動・時刻合わせ機能**:
  - 面倒な時刻設定は不要です。PCから本プログラムを書き込む際、コンパイルマクロを通じて**書き込みを行ったMac/PCの現在時刻にRTCの時間が全自動で同期**されます。
- **日本語ステータス確認画面**:
  - 普段は電力を節約するために画面は完全に消灯（輝度0）しています。
  - 本体のフロントボタン（BtnA）またはサイドボタン（BtnB）を押したときだけ、**15秒間だけ画面が点灯**し、以下の情報を日本語で確認できます：
    - 現在の正確な日付と時刻
    - 電源供給状態（USB給電中 または 内蔵バッテリー残量%）
    - 現在の炎エフェクトモード（通常炎 / 魂の青い炎）
    - 動作時間内の判定状況（動作時間内 / スリープ優先）
- **炎ゆらぎエフェクト & 滑らかなフェード**:
  - パーリンノイズによる自然な炎のゆらぎ（オレンジ系 / 青系）を表現。
  - 点灯開始時と消灯時に**滑らかなフェードイン・フェードアウト（500ms）**を行うことで、高級感のあるライティングを演出します。

## 開発環境のセットアップ (macOS)

本プロジェクトをビルド・書き込みするためには、軽量で強力な **PlatformIO Core (CLI)** を使用します。

### 1. PlatformIO Core のインストール
Macのターミナルを起動し、以下のコマンドを実行して PlatformIO Core を公式スクリプトから安全にインストールします（Python3がインストールされている必要があります）。

```bash
# インストーラーのダウンロードと実行
curl -fsSL https://raw.githubusercontent.com/platformio/platformio-core-installer/master/get-platformio.py -o get-platformio.py
python3 get-platformio.py

# インストール完了後、一時ファイルを削除
rm get-platformio.py
```

### 2. コマンドへのパス設定
ターミナルから `pio` コマンドを直接使えるようにするため、シェル（zsh）の設定ファイルにパスを追加します。

```bash
# zsh設定ファイルにパスを追加
echo 'export PATH="$HOME/.platformio/penv/bin:$PATH"' >> ~/.zshrc

# 設定を現在のターミナルに反映
source ~/.zshrc
```

正常にインストールされたか確認するため、以下のコマンドを実行してバージョンが表示されれば準備完了です。
```bash
pio --version
# 出力例: PlatformIO Core, version 6.1.19
```

## 使用方法

### 1. デバイスの接続と書き込み
M5StickC Plus2 をUSBケーブルでMacに接続し、プロジェクトのルートディレクトリで以下のコマンドを実行します。ビルドとデバイスへの書き込みが同時に実行されます。

```bash
# 書き込み (Macの現在時刻が自動的にM5に書き込まれます)
pio run -t upload
```

### 2. 設定のカスタマイズ
`src/main.cpp` の以下の定数や関数を変更することで、動作を調整できます。

```cpp
#define NUM_LEDS 37         // LEDの個数
#define LED_BRIGHTNESS 170  // 最大の明るさ (0〜255)
const unsigned long KEEP_ALIVE_MS = 10000; // 点灯維持時間 (ミリ秒)

// 動作時間帯の変更 (isWithinActiveHours 関数)
bool isWithinActiveHours() {
    auto dt = M5.Rtc.getDateTime();
    int hour = dt.time.hours;
    return (hour >= 21 || hour < 5); // 21:00 〜 翌朝 5:00
}
```

### 3. 操作方法
- **人感センサー検知**: 21:00〜5:00の間、センサーが動きを感知するとLEDがフェードイン点灯します。検知がなくなってから10秒後にフェードアウト消灯し、自動的に Light Sleep（超省電力）に入ります。
- **サイドボタン (BtnB) またはフロントボタン (BtnA) 押下**: 画面が15秒間オンになり、現在時刻と日本語の動作ステータスを確認できます。
- **フロントボタン (BtnA) 押下（画面ON時）**: LEDのカラーモードが切り替わります。
  - **ゆらぎ炎 (通常ランタン)**: 暖かみのあるオレンジ・黄色の炎。
  - **青い炎 (魂のランタン)**: マインクラフトのソウルランタンのような幻想的な青色の炎。

## 省電力動作の注意点（モバイルバッテリー使用時）
「動作時間外（昼間）」や「LED消灯時」は、M5StickC Plus2が非常に消費電力の低い Light Sleep 状態になります。
一般的なモバイルバッテリーは、消費電流が極端に低くなると**自動的に給電を停止（電源オフ）**してしまうことがあります。
その場合は、**「低電流モード（イヤホン充電モード等）」を搭載したモバイルバッテリー**をご使用いただくか、**USB-ACアダプタ等でコンセントから給電**してご利用ください。
